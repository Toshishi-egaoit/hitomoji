# インストール手順(v0.1.4）

このソフトウェアをインストールするには、以下の手順に従ってください。

- 後悔されている以下のリポジトリからソースコードをクローンします。
  git clone
- VS022でソリューション(hitomoji.sln）を開き、全ビルドを実行します。
　（３つのプロジェクトが存在します）
- ビルドが成功したら、出力フォルダ（x64/Debug)に移動します。
- ここに、以下の２つのファイルがあることを確認します。
　-- hitomoji.dll
  -- registerApp.exe
- 管理者権限でコマンドプロンプトを開き、registerApp.exe を実行します。
- スタートメニューから、設定アプリを起動します。
　「時刻と言語」→「言語と地域」→「日本語（言語のオプション）」を選択します。
　「インストールされているキーボード」の「キーボードを追加する」をクリックし、
　「ひともじ」を選択します。

以上で、「ひともじ」が使えるようになります。
MS-IMEとの切り替えは、タスクバーのIMEアイコンから行えます。
